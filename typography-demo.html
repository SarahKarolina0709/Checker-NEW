Typography System Demo

Demonstration des vereinheitlichten Typography-Systems basierend auf der Translation Quality GUI.

Page Title - Haupttitel der Anwendung .ty-title • 26px bold • line-height: 1.2

Verwendung: Hero-Titel, Hauptseiten-Überschriften, wichtigste Hierarchie-Ebene

Section Heading - Haupt-Sektionsüberschrift .ty-heading • 22px bold • line-height: 1.2

Verwendung: Hauptüberschriften, wichtige Sektionen, primäre Inhalts-Hierarchie

Card Header - Card- und Sub-Sektions-Titel .ty-subheading • 18px bold • line-height: 1.2

Verwendung: Card-Header, Sub-Sektionen, sekundäre Inhalts-Hierarchie

Body Text Regular - Standard-Textgröße für Inhalte .ty-body • 14px normal • line-height: 1.4

Dies ist der Standard-Fließtext der Anwendung. Optimiert für Lesbarkeit bei längeren Textpassagen. Verwendet für Beschreibungen, Eingabefelder und allgemeine Inhalte.

Body Text Bold - Hervorgehobener Body-Text .ty-body-bold • 14px bold • line-height: 1.4

Verwendung: Button-Text, wichtige Labels, hervorgehobene Inhalte

Caption Text - Kleine Labels und Zusatzinformationen .ty-caption • 12px normal • line-height: 1.4

Verwendung: Menü-Text, Status-Anzeigen, Hilfs-Texte, Metadaten

Interactive Elements

Alle Buttons verwenden .btn Klasse mit ty-body-bold (14px bold)

Form Elements

Meta-Information: Verwendet caption für zusätzliche Hinweise

Typography Hierarchy

1. Title (26px) - Page Level
2. Heading (22px) - Section Level
3. Subheading (18px) - Card Level
4. Body Bold (14px) - Emphasis
5. Body (14px) - Content
6. Caption (12px) - Meta

CSS Variables Reference

/* Direct CSS Variable Usage */
.custom-title {
    font-size: var(--ty-title-size);        /* 26px */
    font-weight: var(--ty-title-weight);    /* 700 */
    line-height: var(--ty-title-line-height); /* 1.2 */
}

.custom-button {
    font-size: var(--ty-body-bold-size);    /* 14px */
    font-weight: var(--ty-body-bold-weight); /* 700 */
}
